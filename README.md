# yii2-widgets

Collection of useful Widgets for Yii Framework 2.0

---

## Installation

In yii2 composer.json, additional require yidas/yii2-helpers.
```
"require": {
        ...
        "yidas/yii2-widgets": "*"
    },
```

Or, install from composer command:
```
$ php composer.phar require yidas/yii2-widgets
```
